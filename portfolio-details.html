Arabic RAG System for 30K PDF Knowledge Base

Developed a Retrieval-Augmented Generation (RAG) system capable of processing and answering queries from over 30,000 Arabic PDF files. Implemented advanced preprocessing, chunking, and semantic search using FAISS/ChromaDB. Integrated with OpenRouter API for LLM inference and deployed a professional Streamlit interface with persistent chat history.

Tech Stack: Python, LangChain, FAISS, ChromaDB, OpenRouter API, Streamlit, FastAPI, SQLite, Docker, Cloud Hosting.
